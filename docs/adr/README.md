# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for the Stellar Portfolio Rebalancer project.

## What is an ADR?

An Architecture Decision Record is a short document that captures an important architectural decision made during the project, along with its context and consequences.

## How to Create an ADR

1. Copy \`adr-template.md\` and name it \`NNNN-title-with-dashes.md\`
2. Fill in the sections
3. Submit as a pull request
