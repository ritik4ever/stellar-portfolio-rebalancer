# Architecture Decision Records (ADR)

This folder documents architecture decisions made for the Stellar Portfolio Rebalancer.

## How to Add an ADR

1. Copy `docs/adr/TEMPLATE.md` to `docs/adr/000N-title.md` (increment N)
2. Fill in the template sections
3. Update this README with a link to the new ADR
4. Include the ADR in your PR for review
