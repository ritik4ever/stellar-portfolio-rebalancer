# Wallet Connection & Signature FAQ

## Common Issues

### Freighter Wallet Not Detected
**Problem:** Freighter extension is installed but not detected.
**Solution:** Ensure Freighter is installed and unlocked. Refresh the page. Check browser permissions.

### Signature Request Timeout
**Problem:** Transaction signature request times out.
**Solution:** Make sure Freighter is unlocked and you approve the signature within the timeout window. Check network connection.

### Invalid Network
**Problem:** Transaction fails with network error.
**Solution:** Ensure you are connected to the correct Stellar network (Testnet vs Mainnet). Switch in Freighter settings.

### Insufficient Balance
**Problem:** Transaction fails with insufficient funds.
**Solution:** Ensure your wallet has enough XLM for the base reserve and transaction fees.

### Wrong Account
**Problem:** Signature from wrong Stellar account.
**Solution:** Switch to the correct account in Freighter before signing.

## Troubleshooting Steps

1. Open Freighter extension and verify it's unlocked
2. Check network selection (Testnet/Mainnet)
3. Refresh the dApp page
4. Clear browser cache and reconnect
5. Check browser console for errors (F12 → Console)
