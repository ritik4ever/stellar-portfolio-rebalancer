# Glossary

## Stellar Network Terms

| Term | Definition |
|------|------------|
| **Stellar** | Decentralized payment network for cross-border transactions |
| **XLM** | Native cryptocurrency of the Stellar network (Lumens) |
| **Account** | Stellar public key starting with G... |
| **Secret Key** | Private key starting with S... used to sign transactions |
| **Sequence Number** | Monotonically increasing number for transaction ordering |
| **Base Reserve** | Minimum XLM balance required to fund an account (0.5 XLM) |
| **Trustline** | An asset holder's declaration of trust in a specific asset issuer |

## Soroban Terms

| Term | Definition |
|------|------------|
| **Soroban** | Stellar's smart contract platform |
| **Contract** | WASM-based program deployed on Soroban |
| **WASM** | WebAssembly binary format for Soroban contracts |
| **Invoke** | Calling a contract function via a Soroban transaction |
| **Host Function** | Built-in functions provided by the Soroban runtime |
