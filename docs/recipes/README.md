# API Curl Recipe Examples

This folder contains practical `curl` command examples for common API tasks.

## Prerequisites

- `curl` installed on your system
- A running instance of the portfolio rebalancer backend (default: `http://localhost:3000`)

## Recipes

- [Rebalancing](rebalancing.md) — Trigger and monitor rebalancing operations
- [Portfolio](portfolio.md) — Manage portfolios and track allocations
- [Health](health.md) — Check API and service health

## Usage

```bash
# Run any recipe file directly
bash docs/recipes/rebalancing.sh
```
