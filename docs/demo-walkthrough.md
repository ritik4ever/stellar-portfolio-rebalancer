# Demo Walkthrough

This walkthrough guides you through the key features of Stellar Portfolio Rebalancer.

## 1. Connecting Your Wallet

1. Install the Freighter browser extension
2. Create or import a Stellar wallet
3. Click "Connect Wallet" on the top-right corner
4. Approve the connection request in Freighter

## 2. Viewing Your Portfolio

- The dashboard shows your current portfolio composition
- Each asset displays balance, percentage allocation, and value
- Use the summary cards for a quick overview

## 3. Creating a Rebalance Strategy

1. Navigate to the Rebalance page
2. Select a strategy (Equal Weight, Target Allocation, or Custom)
3. Set your desired allocation percentages
4. Click "Preview Rebalance" to see the proposed changes

## 4. Executing a Rebalance

1. Review the transaction details in the preview
2. Verify the swap amounts and expected output
3. Click "Execute" and sign the transaction in Freighter
4. Wait for confirmation on the Stellar network

## 5. Monitoring Transactions

- The History tab shows all past rebalance transactions
- Each entry shows status, timestamp, and details
- Click any transaction for full information
