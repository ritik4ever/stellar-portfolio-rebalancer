# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ |

## Reporting a Vulnerability

Please report security issues privately to the maintainers:

1. **Email**: Send details to the project maintainer (see CODEOWNERS)
2. **Do not** open a public GitHub issue
3. Include: affected version, description, reproduction steps, and impact

You can expect:
- Acknowledgment within 48 hours
- Regular updates on progress
- Credit in release notes (if desired)

## Disclosure Policy

- We will acknowledge receipt within 2 business days
- A fix will be prioritized based on severity
- We will notify users via GitHub Security Advisories
- Public disclosure after a fix is released (typically within 14 days)
